Importante considerar que nos exemplos, eu criei um package chamado geek, ou seja, se for utilizar algum script desses em seu computador, se atente ao package que irá utilizar.

Script1.java
Faça um algoritmo para somar dois numeros e multiplicar o resultado pelo primeiro numero.

Script2.java
Faça um algoritmo para "calcular o estoque médio de uma peça", sendo que:
estoque_medio = (quantidade_minima + quantidade_maxima)/2

Script3.java
Faça um algoritmo que peça dois numeros e imprima a soma.

Script4.java
Faça um algoritmo que converta metros para centímetros

Script5.java
Faça um algoritmo que pergunte quanto você ganha por hora e o numero de horas trabalhadas no mes. Calcule e mostre o total do seu salário no referido mes.

Script6.java
Tendo como dados de entrada a altura de uma pessoa, construa um algoritmo que calcule seu peso ideal, usando a seguinte formula: (72.7 * altura) - 58
